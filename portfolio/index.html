Operating Systems Projects

Virtual Memory Management

Demand Paging is a virtual memory management mechanism that would only load and map pages from persistent storage to memory when needed, i.e., when a process tries to access a page that is not residing in the physical memory (which would result in a page fault). The OS will then try to find a free (unoccupied) physical frame, load, and map the page to the free physical frame as part of the page fault handling.
